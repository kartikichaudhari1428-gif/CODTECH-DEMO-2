# CODTECH-DEMO-2

This project demonstrates a simple Natural Language Generator (NLG) using random word selection.
It generates sentences by combining subjects, verbs, objects, and adverbs.
The goal is to help beginners understand how computers can create human-like text using basic programming concepts.
